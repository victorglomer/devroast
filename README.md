# Devroast

Brutally honest code reviews powered by AI.

## O que é o Devroast?

O Devroast é uma plataforma onde você pode submeter seu código e receber uma avaliação — sendo bem honesto sobre os pontos negativos. É inspirador no estilo "Hall da Vergonha" do GitHub, mas com pitadas de humor e sarcasmo.

## Funcionalidades

- **Submeter código** — Cole seu código e receba uma nota de 0 a 10
- **Modo Roast** — Escolha entre uma avaliação normal ou no modo "roast" (máximo sarcasmo)
- **Leaderboard** — Veja os códigos com pior nota e se inspire a melhorar (ou não)
- **Syntax highlighting** — Código formatado com cores para melhor leitura

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- TypeScript
- Base UI (componentes acessíveis)
- Shiki (syntax highlighting)

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Acesse [http://localhost:3000](http://localhost:3000) para usar o app.
